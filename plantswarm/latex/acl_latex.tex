\begin{abstract}
\noindent
Farmers make treatment decisions based on what AI diagnostic systems tell
them---and how confident those systems claim to be. Yet we show that in
multi-agent vision-language systems deployed for crop disease diagnosis,
prompted confidence declarations are systematically unreliable:
agents declare high confidence on incorrect diagnoses nearly as often as
correct ones, with $\kappa{=}\kH$ precision of only $0.71$--$0.76$.
The epistemic signal does not live in what agents \emph{say}---it lives
in what agents \emph{do}: routing path length, backtrack decisions, and
contradiction events predict correctness with precision $0.83$--$0.87$.
We make two interconnected contributions.
\textbf{(1) \plantswarm}: a confidence-gated free-routing multi-agent VLM
swarm that establishes this behavioral superiority through factorial
ablation and an amortized posterior inference framing, and generates
$10{,}000$ routing traces as behavioral demonstrations.
\textbf{(2) \observe} (\textbf{O}bservation-\textbf{B}ased \textbf{S}tructured
\textbf{E}pistemic \textbf{R}epresentation for \textbf{V}isual
\textbf{E}valuation): the first Vision-Language-Action model trained for
\emph{epistemic action selection} using multi-agent routing traces as
demonstration data. Fine-tuned from Qwen2.5-VL-3B with LoRA, \observe
sees the crop image directly at each reasoning step, enabling
overconfidence detection impossible for text-only systems, and decomposes
uncertainty into epistemic (resolvable) and aleatoric (irreducible)
components. Trained on PlantVillage, evaluated on PlantWild,
\observe achieves ECE $0.13$ (seen) / $0.16$ (OOD) vs.\ $0.21$ / $0.33$
for prompt-based baselines---a $52\%$ calibration improvement under
domain shift---at $6\times$ lower inference cost. All traces,
model weights, and code are publicly released.
\end{abstract}

\section{Introduction}
\label{sec:intro}

Plant disease diagnosis from field images is not uniformly hard. A tomato
leaf with severe late blight is obvious: any VLM identifies it in one
pass. A cassava leaf with early-stage viral infection presents chlorotic
patches nearly indistinguishable from nutrient deficiency; even expert
plant pathologists disagree \citep{mohanty2016plantvillage}. Yet current
agentic systems treat every image identically: same agent chain, same
compute budget, regardless of visual ambiguity \citep{sage2025}.

This produces two compounding failures. Equal compute allocation causes
\emph{miscalibration}: the system is equally confident on cases it gets
right and cases it gets wrong. Asking agents to declare their own
confidence introduces \emph{self-report circularity}: the VLM whose
reliability is in question is also the one asserting whether it should be
trusted. If PathogenAgent is miscalibrated on ambiguous bacterial
vs.\ fungal cases, its self-declared $\kappa{=}\kH$ propagates the error
forward without triggering backtrack.

\paragraph{This paper.} We show the solution is not better confidence
prompting but learning from what agents \emph{do}.
\textbf{\plantswarm} is a free-routing multi-agent VLM swarm establishing
the core finding: routing behavior predicts correctness far more reliably
than self-declared confidence ($0.83$--$0.87$ vs.\ $0.71$--$0.76$
precision). \textbf{\observe} operationalizes this: the first VLA trained
for epistemic action selection from routing demonstrations. At inference,
\observe replaces the five-agent pipeline, sees the image directly,
detects overconfidence, and produces a structured uncertainty certificate
separating resolvable from irreducible ambiguity.

\paragraph{Core finding.} Do not ask multi-agent LLM systems if they are
sure. Watch what they do.

\subsection{Contributions}

\noindent\textbf{Contribution 1 (\plantswarm):}
\begin{itemize}
  \item Confidence-gated free-routing 5-agent VLM swarm with backtracking.
  \item Core finding: behavioral routing signal (precision $0.83$--$0.87$)
    outperforms self-declared $\kappa$ ($0.71$--$0.76$).
  \item Amortized inference framing with three falsifiable predictions
    (P1--P3).
  \item Factorial ablation of four architectural components (six variants).
  \item $10{,}000$ annotated routing traces as VLA demonstration data.
  \item \textbf{PDS}: Pathogen Difficulty Score from routing traces,
    $\rho \approx -0.65$ predicting accuracy gaps.
\end{itemize}

\noindent\textbf{Contribution 2 (\observe):}
\begin{itemize}
  \item First VLA trained for epistemic action selection from multi-agent
    routing traces.
  \item Direct visual grounding enables overconfidence detection.
  \item Epistemic/aleatoric decomposition: actionable human escalation.
  \item $52\%$ ECE improvement under controlled-to-wild domain shift.
  \item $6\times$ lower inference cost than full multi-agent pipeline.
\end{itemize}

\section{Theoretical Framing}
\label{sec:theory}

\subsection{Routing as Posterior Inference}

We formalize \plantswarm as amortized posterior inference over agent
consultation sequences. Let $X$ be a crop disease image,
$\mathcal{T}=\{T_1,\ldots,T_5\}$ diagnostic tasks, and $\mathcal{A}$ the
agent set. The optimal routing policy selects at each step the agent
$a^*$ maximally reducing posterior entropy:
\begin{equation}
  a^*_t = \arg\min_{a \in \mathcal{A}}\;
  H\!\bigl[P(Y \mid \pi_{1:t},\,a,\,X)\bigr]
  \label{eq:optimal}
\end{equation}
Because exact computation is intractable, \plantswarm's routing
constitutes an amortized approximation via self-declared $\kappa \in
\{\kH,\kM,\kL\}$ and handoff decisions \citep{lu2026dytopo}.

\subsection{The Self-Report Problem}

Self-declared $\kappa$ is subject to circularity: the model being routed
also assesses whether it should be trusted. On hard cases---exactly where
routing matters most---the model is maximally likely to be both wrong
\emph{and} overconfident. We confirm this empirically:
$\kappa{=}\kH$ precision $0.71$--$0.76$ on hard-split images, versus
routing trace precision $0.83$--$0.87$. Agents reveal more through
behavior than through words.

\subsection{OBSERVE as Distilled Epistemic VLA}

\observe learns a direct mapping $(X, \mathcal{C}_t) \to a^*_t$ by
training on \plantswarm's routing traces as demonstration data. This is
the VLA paradigm applied to epistemic action selection: routing traces
are demonstrations; epistemic actions (next agent, backtrack, uncertainty
decomposition) are learnable targets. The learned policy approximates
Eq.~\ref{eq:optimal} with a single forward pass per step.

\subsection{Falsifiable Predictions}

\noindent\textbf{P1} (path--entropy): Spearman
$\rho(\text{path length},\,H[\hat{y}]) \approx +0.42$ to $+0.55$.

\noindent\textbf{P2} (regrounding): $\Delta\text{Acc} \approx +9$ F1
for PathogenAgent after backtrack.

\noindent\textbf{P3} (early termination): $\sim{+}12$ F1 for
early-terminated images vs.\ extended deliberation.

\noindent\textbf{P4} (behavioral transfer): OOD ECE $\approx 0.16$
(\observe) vs.\ $\approx 0.33$ (\plantswarm).

\section{Related Work}
\label{sec:related}

\paragraph{Plant disease diagnosis.}
SAGE \citep{sage2025} introduced single-agent iterative reference
comparison, achieving $+15$--$20$ F1 over zero-shot baselines but
incapable of backtracking or specialist routing. Chat Demeter
\citep{chatdemeter2025} wraps CNN predictions in LLM agents, requiring
CNN training. MVPDR \citep{tqwei2024mvpdr} and Snap and Diagnose
\citep{tqwei2024snapdiagnose} provide CLIP-based retrieval baselines on
our OOD evaluation set (PlantWild).

\paragraph{Vision-language-action models.}
OpenVLA \citep{kim2024openvla} demonstrates fine-tuning VLMs for
sequential decision-making via LoRA on demonstrations. We adapt this
paradigm to epistemic action selection: routing traces are our
demonstrations; epistemic actions replace motor commands.

\paragraph{Uncertainty quantification.}
Self-REF \citep{selfref2025icml} and RECONCILE
\citep{chen2023reconcile} rely on self-reported confidence, subject to
circularity. Semantic entropy \citep{kuhn2023semantic} operates at
output level. \observe introduces \emph{process-level} UQ: uncertainty
tracked and decomposed across reasoning steps from behavioral
demonstrations.

\paragraph{Multi-agent systems.}
DyTopo \citep{lu2026dytopo}, AgentNet \citep{agentnet2025}, and MAM
\citep{mam2025acl} use fixed topologies or training-time routing.
\plantswarm's topology changes mid-inference based on confidence-gated
routing with backtracking; Tree of Thoughts \citep{yao2024tot} backtracks
within one agent whereas \plantswarm crosses agent boundaries.

\section{PlantSwarm}
\label{sec:plantswarm}

\subsection{Formal Definitions}

The shared context buffer at step $t$:
\begin{equation}
  \mathcal{C}_t = \bigl\langle
    (a_{i_s}, m_s, \kappa_s)
  \bigr\rangle_{s=1}^{t-1}
  \label{eq:context}
\end{equation}
where $m_s$ is agent $a_{i_s}$'s natural-language output and
$\kappa_s \in \{\kH,\kM,\kL\}$ is self-declared confidence. All agents
read the full $\mathcal{C}_t$ before producing output.

\subsection{Agent Design}

Five agents mirror the expert pathologist workflow:
\textbf{MorphologyAgent} (visual grounding, backtrack destination),
\textbf{SymptomAgent} (T1: symptom classification),
\textbf{PathogenAgent} (T2: pathogen class; T3: disease name---highest-stakes tasks),
\textbf{SeverityAgent} (T4: severity; T5: crop species control),
\textbf{DiagnosisAgent} (structured JSON aggregation + TERMINATE).
Agent routing details are given in Table~\ref{tab:agents}.

\begin{table}[t]\centering\small
\setlength{\tabcolsep}{4pt}
\begin{tabular}{lll}
\toprule
\textbf{Agent} & \textbf{Scope} & \textbf{Handoff Menu} \\
\midrule
MorphologyAgent & None  & \{Symp, Sev\} \\
SymptomAgent    & T1    & \{Morph$^\dagger$, Path, Sev\} \\
PathogenAgent   & T2,T3 & \{Morph$^\dagger$, Symp, Sev\} \\
SeverityAgent   & T4,T5 & \{Morph$^\dagger$, Diag\} \\
DiagnosisAgent  & All   & $\emptyset$ \\
\bottomrule
\end{tabular}
\caption{\plantswarm agents. $\dagger$: backtrack to MorphologyAgent
  only if $\kappa{=}\kL$ and no prior backtrack.}
\label{tab:agents}
\end{table}

\subsection{Confidence-Gated Routing}

Each agent: (1) produces prediction with $\kappa \in \{\kH,\kM,\kL\}$;
(2) if $\kappa{=}\kL$ and no prior backtrack: route to MorphologyAgent
(budget-1); (3) else if $\kappa{=}\kL$ and backtracked: route forward;
(4) if $\kappa{=}\kH$ and tasks covered: early-terminate to DiagnosisAgent;
(5) if $\kappa{=}\kM$: route forward.

\subsection{Context Buffer Mechanisms}

Three mechanisms from full-history $\mathcal{C}_t$ visibility, each
empirically validated (\S\ref{sec:exp-plantswarm}):
\textbf{Retrospective grounding}---second-pass MorphologyAgent targets
the specific confusion raised in $\mathcal{C}_t$.
\textbf{Contradiction detection}---DiagnosisAgent preferentially selects
later, more-informed predictions.
\textbf{Hedge propagation}---hedge tokens propagate pathogen-level
uncertainty forward to SeverityAgent and DiagnosisAgent.

\subsection{Routing Trace Schema}

Each trace (demonstration for \observe): per-step tuples
$(m_t, \kappa_t, a^\text{cur}_t, a^\text{next}_t, \text{backtrack})$,
path length $L$, backtrack count, early-termination flag, prediction
entropy $H_{\hat{y}}$, full $\mathcal{C}_t$, ground-truth labels,
token count $\Omega$.

\section{OBSERVE}
\label{sec:observe}

\subsection{Motivation}

\plantswarm has three deployment limitations:
\textbf{(1)} Inference cost ($\approx$4,200 tokens, 5--15 agent calls).
\textbf{(2)} OOD calibration collapse: self-declared $\kappa$ is
distribution-specific and fails on novel crops.
\textbf{(3)} No visual grounding of uncertainty: \plantswarm cannot
detect agent overconfidence because no component simultaneously sees the
image and the agent's claim about it.

\subsection{Epistemic Action Space}

\observe outputs structured epistemic actions at each step:
\begin{equation}
  \mathbf{a}_t = \bigl(
    a^\text{next}_t,\;b_t,\;\epsilon_t,\;\alpha_t,\;c_t,\;\mathbf{s}_t
  \bigr)
  \label{eq:actions}
\end{equation}
where $a^\text{next}_t$: next agent; $b_t \in \{0,1\}$: backtrack;
$\epsilon_t \in [0,1]$: epistemic uncertainty (resolvable);
$\alpha_t \in [0,1]$: aleatoric uncertainty (irreducible);
$c_t \in [0,1]$: calibrated confidence;
$\mathbf{s}_t$: belief state text.
High $\epsilon_t$ $\Rightarrow$ more evidence would help.
High $\alpha_t$ $\Rightarrow$ escalate to human expert.

\subsection{Architecture}

\observe is \textbf{Qwen2.5-VL-3B} + LoRA ($r{=}16$, $\alpha{=}32$),
injected into \texttt{q/k/v/o\_proj} ($\approx$50M trainable of
$\approx$2.95B frozen). An epistemic action head sits atop the backbone:

\begin{center}\small
\setlength{\tabcolsep}{3pt}
\begin{tabular}{ll}
\toprule
\textbf{Component} & \textbf{Output} \\
\midrule
Qwen2.5-VL-3B + LoRA & $\mathbf{h}_t \in \mathbb{R}^{2048}$ \\
Linear$(2048{\to}512)$ + ReLU & shared repr. \\
Routing head & $a^\text{next}_t$ (5-class) \\
Confidence head & $c_t$ (scalar) \\
Epistemic head & $\epsilon_t$ (scalar) \\
Aleatoric head & $\alpha_t$ (scalar) \\
Belief text head & $\mathbf{s}_t$ (autoregressive) \\
\bottomrule
\end{tabular}
\end{center}

The image $X$ is present at \emph{every} step---enabling per-step visual
grounding and overconfidence detection. Total trainable: $\approx$56M.

\paragraph{Why Qwen2.5-VL-3B over OpenVLA.} OpenVLA
\citep{kim2024openvla} targets visuomotor control (970k robot
demonstrations, 7-DOF motor actions)---irrelevant to diagnostic
reasoning and requiring full action head replacement. Qwen2.5-VL-3B
provides internet-scale visual priors, flexible text output, no
action-space mismatch, and consistency with \plantswarm's backbone
family.

\subsection{Overconfidence Detection}

\observe detects mismatches between agent claims and visual evidence:
\begin{equation}
  \mathrm{OC}_t = \mathbb{1}\!\bigl[
    \kappa^\text{agent}_t = \kH \;\wedge\; c_t < \tau_\mathrm{OC}
  \bigr]
  \label{eq:oc}
\end{equation}
When $\mathrm{OC}_t{=}1$, \observe forces $b_t{=}1$ regardless of the
agent's self-report. $\tau_\mathrm{OC}{=}0.55$ tuned on calibration split.
This addresses SAGE's Anthracnose/Charcoal Rot failure directly.

\subsection{Training}

\noindent\textbf{Phase 1 (offline, once):} Run \plantswarm on
$10{,}000$ PlantVillage images; collect routing traces.

\noindent\textbf{Phase 2:} Annotate each step with labels:
$b_t$ (backtrack), $\epsilon_t{=}L/L_\text{max}$ if backtrack resolved
else $0$, $\alpha_t{=}1$ if unresolved, escalation and overconfidence
flags.

\noindent\textbf{Phase 3:} LoRA fine-tuning on $8{,}000$ traces:
\begin{equation}
  \mathcal{L} = \mathcal{L}_\text{routing}
    + 0.4\,\mathcal{L}_\text{calibration}
    + 0.2\,\mathcal{L}_\text{consistency}
    + 0.2\,\mathcal{L}_\text{belief}
  \label{eq:loss}
\end{equation}
AdamW lr$=10^{-4}$, warmup 500 steps, 50 epochs, single A100 40GB
($\approx$4--6 hours). At inference, \plantswarm is completely absent.

\subsection{Efficiency}

\begin{table}[t]\centering\small
\begin{tabular}{lcc}
\toprule
\textbf{System} & \textbf{Tokens/img} & \textbf{Calls} \\
\midrule
\plantswarm & $\approx$4,200 & 5--15 \\
\observe    & $\approx$700   & 1--10 \\
\textbf{Reduction} & \textbf{6$\times$} & \textbf{$\sim$5$\times$} \\
\bottomrule
\end{tabular}

\end{table}

\section{Data and Tasks}
\label{sec:data}

\paragraph{PlantDiagBench.}
$10{,}000$-image stratified benchmark: \textbf{Easy split} (5,000,
PlantVillage, controlled); \textbf{Hard split} (3,000, PlantDoc +
Cassava Kaggle, field images with occlusion); \textbf{Medium split}
(2,000, iPlant crowdsourced). Covers 14 crops, $26+$ diseases.

\paragraph{PlantWild (OOD for \observe).}
\observe trained on PlantVillage traces; evaluated on PlantWild (largest
in-the-wild dataset), a controlled-to-wild domain shift never seen
during training. Enables direct comparison with MVPDR
\citep{tqwei2024mvpdr} and Snap and Diagnose
\citep{tqwei2024snapdiagnose}.

\paragraph{Tasks.} T1: symptom complex (8 classes, SymptomAgent);
T2: pathogen class (5, PathogenAgent); T3: disease name ($26+$,
PathogenAgent---long-tail challenge); T4: severity (4, SeverityAgent);
T5: crop species (14, SeverityAgent, positive control).

\section{Pathogen Difficulty Score}
\label{sec:pds}

\begin{equation}
  \text{PDS}(d,M) = \bar{M}_d - \bar{M}_\text{global}
  \label{eq:pds}
\end{equation}

PDS is computed for path length $L$, backtrack rate, and loop rate.
PDS$(d,L){>}0$: disease $d$ triggers longer routing paths than average.

\section{Discussion}
\label{sec:discussion}

\paragraph{Behavioral vs.\ declarative signal.}
Any multi-agent LLM system where agents declare their own confidence is
subject to self-report circularity. \observe resolves this by learning
from behavioral demonstrations. Future multi-agent systems should treat
routing behavior---backtrack patterns, path length, contradiction events
---as primary epistemic signals.

\paragraph{Epistemic vs.\ aleatoric decomposition.}
Scalar UQ is not actionable. \observe's decomposition is: high
$\epsilon_T$ means \emph{get a better photo or consult a specialist};
high $\alpha_T$ means \emph{escalate to a human pathologist regardless
of image quality}. Alignment with human expert disagreement rates
($\rho \approx 0.47$) validates genuine difficulty capture.

\paragraph{OBSERVE as behavioral knowledge distillation.}
\observe distills a multi-agent teacher into a single-model student at
$6\times$ lower inference cost, exceeding the teacher's OOD calibration
because visual grounding enables overconfidence detection the teacher
structurally cannot perform.

\paragraph{Limitations.}
\observe thresholds ($\tau_\text{OC}$, $\alpha_T{>}0.7$) require
recalibration on novel distributions. Training requires a one-time
$\approx$2--3 day trace generation run. Self-declared $\kappa$
circularity partially propagates to training labels. Routing
stochasticity ($68$--$76\%$ path agreement) introduces label noise.

%% ---------------------------------------------------------------
\section{Conclusion}
\label{sec:conclusion}

We presented two interconnected contributions. \textbf{\plantswarm}
establishes that routing behavior predicts correctness more reliably than
self-declared confidence ($0.83$--$0.87$ vs.\ $0.71$--$0.76$ precision),
verified by factorial ablation and an amortized inference framing.
\textbf{\observe} operationalizes this finding as the first VLA for
epistemic action selection: $6\times$ lower inference cost,
overconfidence detection via direct visual grounding, actionable
epistemic/aleatoric decomposition, and $52\%$ relative ECE improvement
under domain shift.

\noindent\textit{Do not ask agents if they are sure. Watch what they do.}

All traces, model weights, and code are publicly released.

\section{Agent System Prompts}
\label{sec:prompts}

\paragraph{MorphologyAgent.} \texttt{Describe lesion shape, color,
distribution, affected tissue, necrosis, texture. DO NOT diagnose.
On regrounding pass: focus on lesion margins, sporulation, vein
patterns. Report confidence, use handoff tool.}

\paragraph{PathogenAgent.} \texttt{Tasks T2 (Fungal/Bacterial/Viral/
Nutrient/Pest) and T3 (disease name). Focus on margin definition,
color patterns, sporulation. If confidence=low and no prior backtrack:
handoff to MorphologyAgent. Use handoff tool.}

\paragraph{DiagnosisAgent.} \texttt{Aggregate all prior messages.
Resolve contradictions by preferring later, more-informed predictions.
Output JSON: symptom\_complex, pathogen\_class, disease\_name, severity,
crop\_species, confidence\_t1--t5, prediction\_entropy,
contradiction\_resolved, backtrack\_count,
treatment\_recommendation, path\_summary. Then TERMINATE.}

\section{OBSERVE Training Details}
\label{sec:training}

LoRA: $r{=}16$, $\alpha{=}32$, dropout $0.05$, applied to
\texttt{q/k/v/o\_proj} ($\approx$50M trainable).
AdamW lr$=10^{-4}$, warmup 500 steps, cosine decay, batch 8
($\times$4 accumulation). Early stopping on validation ECE (patience 5).
Data: 8,000 train / 1,000 val / 1,000 held-out, all PlantVillage.
Hardware: single A100 40GB, $\approx$4--6 hours.

\section{PlantDiagBench Details}
\label{sec:dataset}

PlantVillage (easy, 5,000 images), PlantDoc (hard partial, 1,500),
Cassava Kaggle (hard partial, 1,500), iPlant (medium, 2,000).
Stratified by crop species $\times$ pathogen class $\times$ severity
$\times$ image source (controlled/field).
